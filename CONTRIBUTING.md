# Contributing

Thanks for your interest in improving pgconf.pl.

## Reporting issues

Open a [GitHub issue](https://github.com/kloba/pgconf.pl/issues) with a clear description and reproduction steps where applicable.

## Proposing changes

1. Fork the repository and create a feature branch from `main`.
2. Make your changes with focused commits.
3. Open a pull request against `main` describing the change and the motivation.

PRs trigger a preview deployment to Azure Static Web Apps; please verify your change renders correctly in the preview before requesting review.

## Code style

Keep HTML/CSS/JS readable. Prefer small, reviewable diffs.
