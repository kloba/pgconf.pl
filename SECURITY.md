# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in this project:

- **Preferred:** use GitHub's private vulnerability reporting — open the repository's [Security tab](https://github.com/kloba/pgconf.pl/security/advisories/new) and submit an advisory. This keeps the report private until a fix is published.
- **Alternative:** open a [GitHub issue](https://github.com/kloba/pgconf.pl/issues/new) with a clear description and reproduction steps. Do not include secrets, personal data, or exploit code that could be misused.

You will receive an acknowledgement within 72 hours. We will work with you on remediation and coordinate public disclosure once a fix is in place.

## Supported Versions

Only the current `main` branch (the live site) is supported.
